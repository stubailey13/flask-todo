# flask-todo
Todo app built with Flask, Semantic UI and SQLite database.

## Setup
Create project with virtual environment:
```bash
$ python3 -m venv .venv
```

## Activate the environment
```bash
$ . .venv/bin/activate
```

## Install Flask
```bash
$ pip install Flask
$ pip install Flask-SQLAlchemy
```

## Run the app
```bash
$ python app.py
```
